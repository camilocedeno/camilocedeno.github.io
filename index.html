CuSmile

Information Retreival System

In this project, in a team with four other students, we created a website where you could query for uplifting news articles. This project was inspired by the necessity of uplifting news during the COVID-19 pandemic. We gathered our dataset by scraping articles from the "Uplifting News" subreddit and using public APIs. We implemented the query searh function by using TF-IDF vector weights and using an inverted index for efficiency. We also applied dimensionality reduction to the term document matrix in order to get lower dimesnion document vectors since the original term-document matrix was very sparse. Try it out by clicking on the image above!

Fair Clustering

Unsupervised Learning Project

A project completed wiht 3 other students that re-implements a fair clustering algorithm proposed in a recent paper. Fair clustering is clustering while taking into acoount protected attributes. For example, in a hypothetical dataset, we can define gender to be a protected attribute which implies that no cluster should have less than some percent of a specific gender. You can access the code and paper by clicking on the image above!

  • Date: December 2019

Teaching

Introduction to ML

During my sophomore year, I became a TA for INFO1988, a student run course for udnergraduates interested in learning about machine learning. The course was designed for students with no prior experience in rigorous math and statisics. The end of goal of the course is for students to have a high level understanding of what buzz words like "big data" and "machine learning models" mean and how to build a simple machine learning pipeline in python. As a TA, I designed the course content by creating the lecture slides and creating code-alongs with Jupyter Notebooks. As an instructor, I managed the course and taught lectures.
